
# GameLounge Backend

This is the backend for the GameLounge system. Built with Node.js, Express, and SQL Server.




## Prerequisites

- Node.js 18.x or higher
- npm 9.x or higher
- SQL Server


## Project Setup

### Clone the repository
`git clone https://github.com/paupet7/GameLoungeRepo.git GameLoungeRepo-backend
cd GameLoungeRepo-backend
`
### In project root create file .env
```
DB_SERVER=localhost (Server)
DB_NAME= database name
DB_USER=sa datbase username
DB_PASSWORD=YourPassword database password
```
### Install dependencies

`npm install`

### Run the project
`npm run dev`
